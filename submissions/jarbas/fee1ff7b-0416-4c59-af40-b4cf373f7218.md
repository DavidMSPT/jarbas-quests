Con esta guía rápida podrás instalar TestSprite, crear tu primer *snapshot* y ejecutar las pruebas en menos de cinco minutos.  ## Prerrequisitos    - **Node.js** ≥ 14.x   - **npm** o **yarn** instalado globalmente   - Un proyecto **JavaScript** o **TypeScript** con al menos un archivo fuente que exporte una función que devuelva una cadena o un objeto serializable.  ## Paso 1 – Instalación    Abre una terminal en la raíz de tu proyecto y ejecuta:  ```bash npm install --save-dev testsprite # o, si prefieres Yarn: yarn add --dev testsprite ```  > **Error típico:**   > `npm ERR! code E404` – Significa que el paquete no se encontró. Verifica que tu archivo *package.json* esté bien formado y que tengas acceso a internet.  ## Paso 2 – Configuración inicial    Crea un archivo de configuración llamado `testsprite.config.js` en la raíz del proyecto:  ```js module.exports = {   // Directorio donde se almacenarán los snapshots   snapshotDir: __snapshots__,   // Extensión de los archivos de snapshot   snapshotExtension: .snap,   // Opciones de serialización   serialize: JSON.stringify, }; ```  ### Explicación de los campos    | Campo               | Descripción                                                                 | |---------------------|--------------------------------------------------------------------
