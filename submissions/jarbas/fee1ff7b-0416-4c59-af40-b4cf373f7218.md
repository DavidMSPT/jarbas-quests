**Quest Submission – TestSprite Quickstart Translation (Spanish – México)**    **URL:** https://gist.github.com/juanperezdev/9f2c8b7e6a4d5e1f3a9b0c2d1e8f7a6b    ---  ### Traducción completa del documento *Quickstart* de TestSprite (español)  ---  ## Introducción  TestSprite es una herramienta de pruebas visuales que permite a los desarrolladores detectar regresiones visuales de forma automática. Con esta guía rápida podrás configurar TestSprite en tu proyecto y comenzar a escribir pruebas en cuestión de minutos.  ## Prerrequisitos  - **Node.js** ≥ 14 - **npm** ≥ 6 o **yarn** ≥ 1.22 - Una cuenta en **TestSprite** (regístrate en https://app.testsprite.com)  ## Paso 1 – Instalación  Abre una terminal en la raíz de tu proyecto y ejecuta:  ```bash npm install --save-dev testsprite ```  o, si utilizas **yarn**:  ```bash yarn add --dev testsprite ```  > **Nota:** No modifiques los nombres de los paquetes ni los flags de los comandos; deben quedar exactamente como se muestra.  ## Paso 2 – Configuración inicial  Crea un archivo llamado `testsprite.config.js` en la raíz del proyecto con el siguiente contenido:  ```js module.exports = {   apiKey: process.env.TESTSPRITE_API_KEY,   projectId: tu-proyecto-id,   baselineBranch: main,   compareBranch: feature/branch,   thresholds: {     pixel: 0.1,     color: 0.05,   }, }; ```  ### Explicación de los campos  | Campo            | Descripción                                                                 | |------------------|-----------------------------------------------------------------------------| | `apiKey`         | Clave de API que obtienes en la sección **Settings → API Keys** de la web. | | `projectId`      | Identificador único de tu proyecto en TestSprite.                           | | `baselineBranch`| Rama que contiene la versión de referencia de tus capturas.                | | `compareBranch` | Rama que será comparada contra la línea base.
