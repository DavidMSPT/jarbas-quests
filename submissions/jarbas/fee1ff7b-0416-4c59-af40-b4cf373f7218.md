Con este tutorial rápido podrás:  - Crear tu primer proyecto de pruebas. - Configurar un entorno de ejecución. - Ejecutar una prueba simple contra una aplicación web. - Visualizar los resultados en el panel de TestSprite.  > **Requisitos previos**   > - Tener una cuenta en TestSprite (regístrate en https://app.testsprite.com).   > - Tener Node.js (versión ≥ 14) y npm instalados en tu máquina.   > - Conocimientos básicos de JavaScript y de la línea de comandos.  ---  ## Paso 1 – Instalación del CLI  Abre una terminal y ejecuta el siguiente comando para instalar la herramienta de línea de comandos (CLI) de TestSprite globalmente:  ```bash npm install -g testsprite-cli ```  > **Error común:**   > `EACCES: permission denied` – Si encuentras este error, vuelve a ejecutar el comando con `sudo` (en macOS/Linux) o abre la terminal como administrador (en Windows).  Una vez completada la instalación, verifica la versión instalada:  ```bash testsprite --version ```  Deberías ver una salida similar a:  ``` TestSprite CLI v1.2.3 ```  ---  ## Paso 2 – Iniciar sesión en la CLI  Inicia sesión con tu cuenta de TestSprite usando el comando `login`.
