Spin up an Ubuntu 22.04 aarch64 VM, install `ffmpeg` and `nginx‑rtmp-module`.     2. Configure Nginx to accept RTMP streams and pipe them to FFmpeg profiles (1080p‑30 fps, 720p‑30 fps, 480p‑30 fps).     3. Use `aws s3 sync` (or OCI Object Storage CLI) to upload generated HLS segments to the origin bucket.    - **Estimated monthly usage:** ~2,200 CPU hours (≈ 30 % avg. core utilization), 12 GB RAM average.    ---    ### 2.
