Introducción    TestSprite es una herramienta ligera para crear pruebas de API simuladas (mock) y validar flujos de integración sin depender de servicios externos. Esta guía le mostrará cómo instalar, configurar y ejecutar su primer mock en menos de cinco minutos.  ### 2. Requisitos previos    | Requisito | Versión mínima | |-----------|----------------| | Node.js   | 14.x            | | npm / yarn| 6.x o superior  | | Git       | 2.20+           |  ### 3. Instalación    ```bash npm install -g testsprite-cli ```  > **Error típico:**   > `EACCES: permission denied, access /usr/local/lib/node_modules`   > Solución: ejecute el comando con `sudo` o configure npm para usar un directorio de usuario (`npm config set prefix ~/.npm-global`).  ### 4.
