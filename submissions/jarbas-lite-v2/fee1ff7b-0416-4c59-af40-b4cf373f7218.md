Introducción  TestSprite es una plataforma ligera para crear pruebas automatizadas de APIs RESTful sin escribir código. Esta guía rápida le mostrará cómo instalar TestSprite, crear su primer proyecto y ejecutar sus primeras pruebas en menos de 10 minutos.  ### 2. Requisitos previos  - **Node.js** ≥ 14.x (descargue desde https://nodejs.org/) - **npm** (incluido con Node.js) - Acceso a una terminal (CMD, PowerShell, Bash, etc.) - Una cuenta en TestSprite (regístrese en https://app.testsprite.com)  ### 3. Instalación  ```bash npm install -g testsprite-cli ```  > **Mensaje de error típico:**   > `npm ERR! code EACCES` – Solucione los permisos de npm o use `sudo` en Linux/macOS.  Verifique la instalación:  ```bash testsprite --version ```  Debería obtener algo como:  ``` testsprite-cli version 1.2.3 ```  ### 4. Creación de un proyecto  1. **Inicialice** un nuevo proyecto en la carpeta deseada:     ```bash    mkdir mi-prueba-api    cd mi-prueba-api    testsprite init    ```  2. El asistente le pedirá:     - **Nombre del proyecto:** (p.ej., *MiPrimerAPI*)    - **URL base de la API:** (p.ej., `https://api.ejemplo.com/v1`)    - **Formato de salida:** `json` o `html` (seleccione `json` para esta guía)     Al finalizar, se generará un archivo `testsprite.config.json`.  ### 5.
