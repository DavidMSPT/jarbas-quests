**TestSprite – Guía de inicio rápido (Español)**    *Fuente original: https://docs.testsprite.com/quickstart*    ---    # Introducción    TestSprite es una herramienta de pruebas visuales que permite a los desarrolladores detectar regresiones de UI de forma automática. Esta guía de inicio rápido le mostrará cómo instalar TestSprite, crear su primer proyecto y ejecutar sus primeras pruebas en menos de 10 minutos.    ---    # Requisitos previos    - **Node.js** ≥ 14.0 (https://nodejs.org/)   - **npm** o **yarn** instalado   - Un proyecto web o móvil que ya compile con **webpack**, **vite**, **react‑native** u otro bundler.    ---    # 1. Instalación    Abre una terminal en la raíz de tu proyecto y ejecuta:    ```bash npm install --save-dev testsprite # o, si usas yarn yarn add --dev testsprite ```    Esto añadirá `testsprite` a tu `package.json` y descargará los binarios necesarios.    ---    # 2. Configuración inicial    Crea un archivo llamado `testsprite.config.js` en la raíz del proyecto con el siguiente contenido:    ```js module.exports = {   // Ruta donde se guardarán las capturas de pantalla de referencia   referenceDir: ./testsprite/reference,    // Ruta donde se guardarán las capturas de pantalla de las pruebas   outputDir: ./testsprite/output,    // Opciones de comparación de imágenes   diff: {     // Umbral de diferencia aceptable (0‑100)     threshold: 0.1,     // Color del resaltado de diferencias     highlightColor: #ff00ff   },    // Configuración del navegador (puede ser chrome, firefox, safari)   browser: chrome }; ```    > **Nota:** No modifique los nombres de las variables ni los bloques de código; deben permanecer tal cual para que TestSprite los reconozca.    ---    # 3.
