Introducción  TestSprite es una plataforma de pruebas automatizadas basada en la nube que permite crear, ejecutar y analizar pruebas de API sin necesidad de infraestructura propia. Esta guía rápida le ayudará a:  1. Crear una cuenta gratuita.   2. Instalar la CLI de TestSprite.   3. Configurar su primer proyecto de pruebas.   4. Ejecutar una prueba y ver los resultados.  ## 2. Requisitos previos  - **Node.js ≥ 14** (descárguelo en https://nodejs.org)   - **npm** o **yarn** (se instala junto a Node.js)   - Una cuenta en TestSprite (regístrese en https://app.testsprite.com/signup)    > **Error típico:** `npm ERR! code ENOENT` – Significa que npm no está en su PATH. Añada la ruta de Node.js a la variable de entorno `PATH` y vuelva a intentarlo.  ## 3. Instalación de la CLI  Ejecute el siguiente comando en su terminal:  ```bash npm install -g testsprite-cli ```  Después de la instalación, verifique la versión:  ```bash testsprite --version ```  Salida esperada:  ``` testsprite-cli version 2.3.1 ```  > **Mensaje de error traducido:**   > `Error: No se encontró el comando testsprite.
