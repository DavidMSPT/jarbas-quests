# Frontmatter Grow Reddit post + comment karma safely, avoiding bans. Use on a fresh or warmed‑up account to build reputation step‑by‑step.    ## Risk model - **Mass posting identical text** → triggers “spam‑filter: duplicate content” (AutoModerator rule `duplicate_content`).   - **Posting links without context** → flags “spam‑filter: low‑effort link post” (AutoModerator rule `link_without_context`).   - **Rapid comment bursts (≥5 comments/ minute)** → hits “rate‑limit: comment spam” (Reddit API throttle).   - **Using banned keywords or URLs** → activates “spam‑filter: prohibited domain” (AutoModerator rule `prohibited_domains`).   - **Posting in a subreddit where you have <1 karma** → triggers “subreddit‑karma‑requirement” (subreddit’s AutoModerator).   - **Deleting and reposting the same content** → sets off “spam‑filter: repost after delete” (AutoModerator rule `repost_after_delete`).   - **Excessive self‑promotion (>2 self‑promo per week)** → fires “spam‑filter: self‑promo limit” (Reddit policy).   - **Account age <3 days with high posting rate** → raises “new‑account‑spam” flag (Reddit anti‑spam system).  ## Step-by-step: brand‑new account (≤30 days, ≤10 karma) 1. **Day 1‑2:** Join 5 “low‑traffic, high‑karma‑gain” subreddits: `r/AskReddit`, `r/TodayILearned`, `r/ExplainLikeImFive`, `r/NoStupidQuestions`, `r/ChangeMyView`. Set sidebar filters to “no link posts”.   2. **Day 1‑2:** Upvote 20‑30 recent posts in each subreddit (no more than 5 per hour).   3. **Day 2‑3:** Post a *comment* on a top‑ranked post in each subreddit. Use the “Ask a question / add a fact” template (≈30 words, genuine). Do **one** comment per subreddit per day.   4. **Day 4‑5:** Continue commenting, now reply to other comments (≤2 replies per thread). Keep total daily comments ≤8.   5. **Day 6:** Make your **first post** in `r/AskReddit`
