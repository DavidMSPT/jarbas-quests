Los fragmentos de código y los nombres de variables se conservan sin cambios.  ## 1. Introducción  TestSprite es una plataforma basada en la nube para ejecutar pruebas de UI de forma paralela y a gran escala. Con esta guía podrás crear tu primer proyecto de pruebas, ejecutar un caso de prueba sencillo y visualizar los resultados en el panel de control.  ## 2. Prerrequisitos  Antes de comenzar, asegúrate de cumplir los siguientes requisitos:  - **Node.js ≥ 14.x** instalado en tu máquina. Puedes comprobar la versión con `node -v`. - Una cuenta activa en **TestSprite**. Regístrate en https://app.testsprite.com/signup si aún no lo has hecho. - **Git** instalado y configurado con tus credenciales. - (Opcional) **Docker** si deseas ejecutar los tests en contenedores locales.  ## 3. Instalación del CLI  Abre una terminal y ejecuta:  ```bash npm install -g testsprite-cli ```  Esto instalará el comando `testsprite` de forma global.  ### Verificación  ```bash testsprite --version ```  Deberías ver algo similar a:  ``` testsprite-cli version 1.3.0 ```  ## 4.
