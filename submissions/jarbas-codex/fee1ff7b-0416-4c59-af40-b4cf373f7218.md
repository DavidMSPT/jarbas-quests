Esta guía rápida te llevará paso a paso para que puedas comenzar a escribir y ejecutar pruebas automatizadas de UI sin necesidad de conocimientos avanzados de programación. TestSprite es una plataforma basada en la nube que permite crear, ejecutar y analizar pruebas de forma sencilla y colaborativa.  ## Requisitos previos  Antes de iniciar, asegúrate de tener lo siguiente:  - Una cuenta activa en TestSprite (puedes registrarte en https://app.testsprite.com/signup).   - Navegador web moderno (Chrome, Firefox o Edge).   - Acceso a la aplicación web que deseas probar.  ## Paso 1: Crear un nuevo proyecto  1. Inicia sesión en tu cuenta de TestSprite.   2. Haz clic en el botón **“Nuevo proyecto”** en la esquina superior derecha.   3. Introduce un nombre descriptivo para el proyecto, por ejemplo, **“MiPrimerProyecto”** y pulsa **“Crear”**.  ```json {   projectName: MiPrimerProyecto,   description: Proyecto de ejemplo para la guía rápida } ```  ## Paso 2: Añadir un caso de prueba  1. Dentro del proyecto recién creado, selecciona **“Casos de prueba”** en el menú lateral.   2. Pulsa **“Nuevo caso de prueba”**.   3.
