Sólo se ha traducido el texto descriptivo, las etiquetas de la interfaz y los mensajes de error.  ---  ## 1️⃣ Prerrequisitos  Antes de comenzar, asegúrate de cumplir con los siguientes requisitos:  | Requisito | Versión mínima | |-----------|----------------| | Node.js   | 14.x o superior | | npm o Yarn| 6.x o superior | | Git       | 2.20 o superior |  ```bash node -v npm -v   # o yarn -v git --version ```  Si alguna de estas herramientas no está instalada, visita sus páginas oficiales para descargar la versión adecuada.  ---  ## 2️⃣ Crear una cuenta en TestSprite  1. Dirígete a **https://app.testsprite.com**.   2. Haz clic en **Registrarse**.   3. Completa el formulario con tu **correo electrónico**, **nombre de usuario** y **contraseña**.   4. Verifica tu cuenta mediante el enlace que recibirás por correo.  > **Error típico:** `Error: Email already in use` → *El correo electrónico ya está registrado.*  ---  ## 3️⃣ Instalar el SDK de TestSprite  Abre una terminal y ejecuta:  ```bash npm install --save-dev testsprite-sdk ```  Esto añadirá el paquete `testsprite-sdk` a tus dependencias de desarrollo.  ---  ## 4️⃣ Configurar el proyecto  Crea un archivo llamado `testsprite.config.js` en la raíz de tu proyecto con el siguiente contenido:  ```javascript module.exports = {   apiKey: TU_API_KEY_AQUÍ,   projectId: TU_PROJECT_ID_AQUÍ,   // Opcional: especifica entornos   environments: {     development: dev,     production: prod   } }; ```  > **Consejo:** Puedes obtener tu `apiKey` y `projectId` desde el panel de **Configuración** > **API Keys** en la consola de TestSprite.  ---  ##
