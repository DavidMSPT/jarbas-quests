Con esta guía rápida, aprenderás a instalar el SDK, crear tu primer test visual y ejecutar la comparación contra una referencia establecida.  ---    ## Requisitos previos    - **Node.js** ≥ 14.x   - **npm** ≥ 6.x o **yarn** ≥ 1.22   - Un proyecto web con **React**, **Vue** o **Angular** (cualquier framework está bien).   - Una cuenta en [TestSprite.io](https://app.testsprite.com) (registro gratuito).    ---    ## Paso 1 – Instalar el paquete npm    Abre una terminal en la raíz de tu proyecto y ejecuta:    ```bash npm install --save-dev testsprite-sdk ```    o, si prefieres Yarn:    ```bash yarn add --dev testsprite-sdk ```    > **Mensaje de error típico:**   > `npm ERR! code E404` – Significa que el paquete no se ha encontrado. Verifica que el nombre sea `testsprite-sdk` y que tu conexión a internet funcione.    ---    ## Paso 2 – Configurar la clave API    1. Inicia sesión en la consola de TestSprite.   2. Dirígete a **Settings → API Keys** y genera una nueva clave.   3. Crea un archivo llamado `.testspriterc.json` en la raíz del proyecto con el siguiente contenido:    ```json {   apiKey: TU_CLAVE_API_AQUI,   projectId: TU_ID_DE_PROYECTO } ```  > **Etiqueta de la UI:** *Clave API* (API Key)    ---    ## Paso 3
