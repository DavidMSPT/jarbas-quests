Esta guía le mostrará cómo instalar TestSprite, crear su primer proyecto y ejecutar una prueba básica en menos de cinco minutos.  ## Requisitos previos    - **Node.js** ≥ 14.x (descárguelo en https://nodejs.org)   - **npm** o **yarn** (se instala junto con Node.js)   - Acceso a una terminal (CMD, PowerShell, Bash, etc.)    ## Instalación    ```bash npm install -g testsprite-cli ```  > **Nota:** Si prefiere usar Yarn, reemplace `npm` por `yarn` en los comandos siguientes.  Una vez instalado, verifique la versión:  ```bash testsprite --version ```  Debería ver algo similar a:  ``` TestSprite CLI v1.2.3 ```  ## Creación de un nuevo proyecto    Ejecute el siguiente comando para iniciar un proyecto de muestra:  ```bash testsprite init mi-primer-proyecto ```  Esto creará una carpeta llamada `mi-primer-proyecto` con la siguiente estructura:  ``` mi-primer-proyecto/ ├─ testsprite.yml        # Configuración del proyecto ├─ tests/                # Directorio de pruebas │   └─ example.test.js   # Prueba de ejemplo └─ package.json         # Metadatos del proyecto ```  ### Configuración del archivo `testsprite.yml`  ```yaml project:   name: Mi Primer Proyecto   baseUrl: https://jsonplaceholder.typicode.com   timeout: 5000 ```  - **name**: nombre descriptivo del proyecto.   - **baseUrl**: URL base contra la que se eje
