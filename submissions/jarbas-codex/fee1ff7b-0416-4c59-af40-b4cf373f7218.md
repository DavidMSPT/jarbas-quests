**TestSprite – Guía de inicio rápido (Español)**    ---    # Introducción   TestSprite es una plataforma de pruebas visuales que permite a los desarrolladores crear, ejecutar y mantener pruebas de UI de forma sencilla y fiable. Esta guía de inicio rápido le mostrará cómo configurar su proyecto, escribir su primera prueba y ejecutarla en su entorno local.  ---    ## 1. Prerrequisitos    - **Node.js** ≥ 14   - **npm** o **yarn** instalado   - Un proyecto web (React, Vue, Angular, etc.) ya existente   - Cuenta en [TestSprite](https://app.testsprite.com)    ---    ## 2. Instalación    Ejecute el siguiente comando en la raíz de su proyecto para instalar el paquete de TestSprite:    ```bash npm install --save-dev testsprite # o yarn add --dev testsprite ```    Una vez instalado, inicialice la configuración ejecutando:    ```bash npx testsprite init ```    Esto crea un archivo **testsprite.config.js** con la siguiente configuración por defecto:    ```js module.exports = {   baseUrl: http://localhost:3000,   screenshotsFolder: ./testsprite/screenshots,   diffFolder: ./testsprite/diffs,   threshold: 0.1, }; ```    ---    ## 3.
