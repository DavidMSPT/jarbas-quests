Prerrequisitos  - **Node.js** ≥ 14.x   - **npm** ≥ 6.x o **yarn** ≥ 1.22   - Una cuenta en **TestSprite** (crea una en https://app.testsprite.com/signup)    ## 2. Instalación del SDK  Abre una terminal y ejecuta:  ```bash npm install @testsprite/sdk # o, si prefieres Yarn yarn add @testsprite/sdk ```  Esto añadirá el paquete `@testsprite/sdk` a tu `package.json` y descargará las dependencias necesarias.  ## 3. Configuración del proyecto  Crea un archivo llamado `testsprite.config.js` en la raíz de tu proyecto:  ```js module.exports = {   apiKey: TU_API_KEY_AQUI,   projectId: TU_PROJECT_ID_AQUI,   environment: development, // Cambia a production cuando despliegues }; ```  > **Importante:** Sustituye `TU_API_KEY_AQUI` y `TU_PROJECT_ID_AQUI` por los valores que aparecen en el panel de control de TestSprite bajo *Settings → API*.  ## 4. Inicializar el SDK  En el punto de entrada de tu aplicación (por ejemplo, `src/index.js`) importa e inicializa el SDK:  ```js import TestSprite from @testsprite/sdk; import config from ../testsprite.config;  TestSprite.init(config); ```  Esto conectará tu aplicación con el proyecto de TestSprite y habilitará la captura automática de errores y métricas.  ## 5.
