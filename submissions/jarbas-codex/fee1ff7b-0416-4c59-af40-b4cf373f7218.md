Introducción  TestSprite es una herramienta de pruebas basada en la nube que permite crear, ejecutar y analizar pruebas automatizadas de APIs sin necesidad de escribir código complejo. Esta guía rápida le mostrará cómo iniciar un proyecto, crear su primera prueba y visualizar los resultados.  ## 2. Prerrequisitos  - **Cuenta de TestSprite** – Regístrese en https://app.testsprite.com y obtenga una clave API. - **Node.js (v14 o superior)** – Descárguelo desde https://nodejs.org. - **npm o Yarn** – Gestor de paquetes que viene con Node.js. - **Git** – Opcional, para clonar el repositorio de ejemplo.  ## 3. Instalación del CLI  Abra su terminal y ejecute:  ```bash npm install -g testsprite-cli ```  > **Error típico:**   > `EACCES: permission denied, access /usr/local/lib/node_modules`   > **Solución:** Use `sudo` o configure npm para instalar paquetes globales sin privilegios de administrador.  ## 4. Configuración del proyecto  1. **Crear una carpeta de proyecto**     ```bash    mkdir mi-prueba-testsprite    cd mi-prueba-testsprite    ```  2. **Inicializar el proyecto TestSprite**     ```bash    testsprite init    ```     Se le solicitará introducir su **clave API** y el **entorno** (por ejemplo, `sandbox` o `production`). La configuración se guarda en `testsprite.config.json`.     ```json    {      apiKey: TU_CLAVE_API,      environment: sandbox    }    ```  ## 5.
